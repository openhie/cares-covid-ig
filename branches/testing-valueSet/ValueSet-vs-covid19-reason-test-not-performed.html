Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include codes from http://snomed.info/sct where concept is-a 441510007 (Blood specimen with anticoagulant)
  • Include codes from http://snomed.info/sct where concept is-a 123840003 (Sample contaminated)
  • Include codes from http://snomed.info/sct where concept is-a 397933008 (Equipment error/failure (finding))
  • Include codes from http://snomed.info/sct where concept is-a 281264009 (Inappropriate bottle or container for sample)
  • Include codes from http://snomed.info/sct where concept is-a 281268007 (Insufficient sample)
  • Include codes from http://snomed.info/sct where concept is-a 281265005 (Sample incorrectly labeled (finding))
  • Include codes from http://snomed.info/sct where concept is-a 419182006 (Supplies not available (finding))
  • Include all codes defined in http://openhie.org/fhir/covid19-casereporting/CodeSystem/cs-other
